# Todo (CLI)

> Command-line tool to manage the Todo lists

# Create a Todo App

## Requirement Commands

* Add a todo
* Remove a todo
* Check a todo
* Uncheck a todo
* List all todo

### Coding style
*Make sure the code is readable, either by refactoring into `Controller` class or function.

## Additional Commands
* Toggle a todo
* Search todo
